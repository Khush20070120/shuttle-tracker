# Shuttle Tracker System

A full-stack web application built using Flask and MySQL.

## Features
- Login / Signup (Admin & User)
- Bus & Route Management
- Combined Data Table
- Search & Filter
- Analytics Dashboard
- AI Delay Prediction

## Tech Stack
- Python (Flask)
- MySQL
- HTML, CSS, JS
- Chart.js
- scikit-learn

## Run Locally

```bash
pip install -r requirements.txt
python app.py
